QUOI FAIRE AU JARDIN EN MARS

FLEURS :

  • Vous pouvez commencer les boutures de chrysanthèmes, de fuchsias… Placez-les ensuite sous abri légèrement chauffé.
  • Pincez les boutures de géraniums pour que les plants s’épanouissent.
  • Enfin, c’est le moment de divisez les touffes des plantes vivaces telles que asters, phlox…

    • FRUITIERS :

  • Il est encore temps de planter les espèces à fruits rouges : framboisiers, groseilliers, cassissiers. Et, après les dernières gelées, vous pouvez bouturez le figuier et le grenadier ; et à la fin du mois, vous pouvez également commencer les premières greffes en fente sur le cerisier.

    • ARBUSTES :

  • Taillez la glycine ; enlevez les rameaux morts, et palissez les autres.
  • Terminez les plantations des arbustes de terre de bruyère (hortensias macrophyllas, azalées, camelias, rhododendrons, magnolias.....mais aussi sarcoccocas, daphnés, kalmia,....)

    • HAIE :

  • Taillez toutes les espèces ; en coupant les vieux rameaux, vous favoriserez les nouvelles pousses.
  • Terminez les plantations des arbustes à feuilles caduques.

    • PELOUSE :

  • Pour les premières tontes, réglez assez haut afin d’éviter qu’il y ait trop d’herbe sous le carter (il est préférable de faire plusieurs passages rapprochés que de risquer le bourrage).

    • JARDIN D’EAU :

  • Divisez les plantes en utilisant la bêche et/ou le couteau avant de replanter. Plantez les fougères, les primevères…. Bref les plantes qui aiment la fraicheur à leur pied.
